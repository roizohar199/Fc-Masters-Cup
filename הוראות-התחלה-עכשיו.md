## 📝 מה אתה צריך לעשות עכשיו? (5 שלבים)

### שלב 1: העתק Public Key לשרת ⏱️ 3 דקות

**פתח PuTTY והתחבר לשרת שלך**

בשרת, הרץ:
```bash
mkdir -p ~/.ssh
chmod 700 ~/.ssh
nano ~/.ssh/authorized_keys
```

**הדבק את ה-Public Key למעלה** (השורה ש ssh-rsa...)  
שמור: `Ctrl+O`, `Enter`, `Ctrl+X`

```bash
chmod 600 ~/.ssh/authorized_keys
```

✅ **בדיקה:** התנתק ונסה להתחבר שוב - צריך להתחבר ללא סיסמה!

---

### שלב 2: הוסף Secrets ל-GitHub ⏱️ 5 דקות

**גש ל:** https://github.com/roizohar199/Fc-Masters-Cup/settings/secrets/actions

**לחץ "New repository secret" והוסף 4 secrets:**

#### Secret 1:
- Name: `VPS_HOST`
- Value: `[ה-IP של השרת שלך מHostinger]`

#### Secret 2:
- Name: `VPS_USER`  
- Value: `fcmaster` (או `root`)

#### Secret 3:
- Name: `VPS_SSH_KEY`
- Value: **[העתק את כל ה-PRIVATE KEY מלמעלה - כולל BEGIN/END]**

#### Secret 4:
- Name: `VPS_PORT`
- Value: `22`

✅ **בדיקה:** תראה 4 secrets ברשימה


## 🆘 בעיות?

### GitHub Actions נכשל?

```bash

pm2 save
```
